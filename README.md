# Online Coding Interview Platform

A real-time collaborative coding interview platform with syntax highlighting and safe code execution.

## Features

- ✅ Create shareable interview links
- ✅ Real-time collaborative code editing
- ✅ Syntax highlighting for JavaScript & Python
- ✅ Safe code execution in browser (WASM)
- ✅ Live user presence indicators

## Tech Stack

- **Backend**: Node.js + Express + Socket.io
- **Frontend**: React + Vite + Monaco Editor
- **Testing**: Jest + Supertest
- **Deployment**: Docker + Render

## Quick Start

### Prerequisites
- Node.js 20+
- npm

### Installation
```bash
# Clone repository
git clone &lt;your-repo-url&gt;
cd coding-interview-platform

# Install dependencies
npm install

# Run both frontend and backend
npm run dev
